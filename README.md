MMC-2190-AltMenu
================

Описание версии 7
================
1. Имеет возможность запуска навигационной программа, аудиоплеера, видеоплеера, есть настройки звука и экрана, программа радио, включения режима парковочной камеры, индикацию температуры + остаток топлива в баке (спасибо antonio-dj), офисные программы (читалки, калькулятор)
2. Позволяет одновременном пользоваться навигацией+радио (в этом режими звуки навигации не воспроизводятся), навигацией+аудиоплеер. 
3. Автоматически прописывает путь к GPS драйверу и драйверу модема (ярлык на рабочем столе).
4. Есть автоматическая заставка-часы
5. Возможно менять стиль оформления - в настройках.
6. Режим запуска по долгому нажатию кнопки "настройка".
7. В приложении для Радио добавил индикацию громкости и возможность отображать логотип текущей станции (в папку Stations ложить бмп размером 100х65 с именем текущей частоты). Радиостанции сохранять так: сканируем, тапаем на текущую частоту, при этом все пресеты выделятся, тапаем по кнопке нужного пресета.
8. Навигационные программы можно располагать в двух локациях:
USB Disk\Navi или
SDMMC\Navi
При запуске навигации скрипт отдает предпочтение USB, затем SD карте, если навигации там нет - выдает информационное сообщение
9. При запуске навитела специально сделал убивание процесса меню. После выхода из навитела меню запустится автоматом.
Это сделал для того чтоб во время езды между переключениями задач навитела и плеера не вылазило еще и меню.
После запуска навитела музыку можно запустить нажатием на кнопку воспроизведения панели управления плеером.
10. Настройки звука и экрана востанавливаются при запуске НьюМеню
11. Ярлык удаления NewMenu на рабочем столе
12. если на USB флешке в корне создать файл explorer.txt, то при долгом нажатии настроек в штатке запустится експлорер - это сделал на всякий случай для непредвиденных ситуаций.

Установка 
================
Распаковываем на СД-карту или USB-накопитель папку INSTALL из скачанного архива NewMenu. пихаем карту в ММС и выходим на рабочий длительным нажатием кнопки "Настройка".
Заходим на сд-шку и запускаем install.exe из папки install.
Будет предложено установить NewMenu - соглашаемся.
В конце установки ММС перезагрузится в штатку.
Для выхода в альтменю удерживать кнопку "настройки" штатной оболочки.


После установки папку Install можно удалять, меню копируется в Staticstore.

Обновление
================
качаем дистрибутив с альтменю
распаковывавем его на SD или USB накопитель
при выключенной ммс вставляем накопитель в разъем и включаем ммс
если ммс была выключена во время работы альтменю, то после включения будет предложено установить новую версию
если ммс была выключена во время работы штатного меню - переходим в альтменю, и также будет предложено установить новую версию
(при обновлении учитывается версия установленного меню и версия на накопителе, так, если они совпадают, предложения обновить меню не будет)

Описнание Highlight от green-74
=================
1) Основной функционал:
Отображение 5-ти показателей:Температура двигателя, температура АКПП, наружная температура, остаток топлива, скорость.
(Панелька не резиновая, поэтому одновременно только 4), при попытке вывести все 5 - наружная температура выводиться не будет.

При выходе значений за пороговые значения меняется цвет иконки (по желанию), меняется шрифт (цвет, размер) значений (по желанию), проигрывается звуковой сигнал (по желанию) (не играет если включено радио (по желанию) )
При возврате в норму также проигрывается звуковой сигнал (по желанию).
По умолчанию все подсветки, смены шрифтов, звуки -отключены. Пороги дефолтные от фонаря, настраивайте под свой вкус.
2) Отдельно добавил пару скринсэйверов с аналогичным панели хайлайтингом под данную панель.

Устанавливается аналогично любому скину.
Содержимое архива распаковать в папку \StaticStore\Menu\Skins, потом выбирать соответствующие скины.
З.Ы. Сначала применять скин Alt_Full, потом заставки, при обратной последовательности вылезет ошибка.

Настройка:
Тап по активной иконке CheckEngine - сброс ошибок, по неактивной - сброс обучения.
Тап по иконке давления масла - скрыть панель
Тап по иконке заряда АКБ вызывает общие настройки, действуют на все индикаторы.
Тап по показаниям любого индикатора - настройки для индикатора. Для изменения порогов - тапнуть по значению в окне настроек.
Тап по иконке спидометра - переключение профиля город/трасса. Пороговые значения и все параметры для города/трассы настраиваются раздельно. Иконки слегка различаются.
